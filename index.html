YUI3 Forms CSS

Simple styling for HTML Form elements.

First, include it on your page

Add the CSS on your page:

<link rel="stylesheet" type="text/css" href="http://yui.yahooapis.com/gallery-2013.01.09-23-24/build/gallerycss-cssform/gallerycss-cssform.css">

Optionally, you can load it through the YUI().use():

    YUI({
        //specify a gallery build
        gallery: 'gallery-2013.01.09-23-24'
    }).use('gallerycss-cssform', function(Y) {
        Y.log('YUI Loaded');     
    });

Horizontal Form

Horizontal forms are great for standard forms that look well-aligned. The labels are right-aligned against the form input controls.

Grouped Inputs

Grouped inputs are great for grouping small sets of text-based input elements. They work well for sign-up forms, and look natural on mobile devices.

Input Sizing

You can size any input element by adding the classnames below. Since these styles just set width, they can be applied to more than just form controls.

Invalid Inputs

Add the required attribute to any form control.

Disabled Inputs

Add the disabled attribute to any form control.

Rounded Inputs

Add the yui3-input-rounded classname to any form control.
